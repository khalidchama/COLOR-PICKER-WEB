How to Use Colorly: Your Simple Guide to Color Codes

Hello there! As the developer behind Colorly, I'm thrilled to show you just how easy it is to use this tool to get the color codes you need for your projects.

Colorly is designed to be intuitive and fast, providing you with accurate Hex and RGB values in a snap. Hereâ€™s a simple guide on how to get started:

Colorly offers two main ways to select your desired color, and both will instantly provide you with the exact codes you're looking for.

Method 1: Using Your System's Color Picker (Quick Selection)

  1. Click the "Pick with System Picker" button:

    You'll see a standard HTML color input field. Click directly on this field, and your operating system's native color selection window will pop up. This is a great option if you want to quickly browse a wide spectrum of colors or if you're familiar with your system's built-in tools.

  2. Select Your Color:

    Use the tools provided by your system (color wheel, eyedropper, palettes) to find the color you desire.

  3. Watch the Magic Happen:

    As soon as you select a color in your system's picker, Colorly's central color swatch will update to match it, and the Hex and RGB code fields below will instantly display the corresponding values.

Method 2: Adjusting with RGB Sliders (Precise Control)

  1. Adjust the Red, Green, and Blue Sliders:

    Below the system picker, you'll find three individual sliders labeled "Red," "Green," and "Blue." Each slider allows you to choose a value from 0 to 255.

    Dragging these sliders will directly mix the color displayed in the central swatch. You'll see the exact numerical value for each color channel update as you drag.

  2. Fine-Tune Your Hue:

    This method is perfect for precise adjustments or for creating a color by directly specifying its RGB components if you already have the numbers in mind.

Getting Your Color Code

  1. View the Codes:

    The Hex Code (e.g., #007BFF) and RGB Code (e.g., rgb(0, 123, 255)) will be automatically displayed in their respective fields.

  2. Copy to Clipboard:

    Simply click the "Copy" button next to the code you want to use. This will copy the text directly to your clipboard, so you can paste it into your CSS stylesheet, design software, or anywhere else you need it.

That's it! Colorly is built to be a fast tool to help you grab those essential color codes whenever you need them.
